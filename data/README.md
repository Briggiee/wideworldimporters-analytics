# Dataset Information

The dataset used for this project was sourced from the **Wide World Importers** sample database backup obtained from **Kaggle**.

Because the original source is a multi-table SQL database backup file, the full raw dataset is not included in this repository.  
Instead, this project documents the analytical workflow used to load the database into SQL, connect Power BI to the SQL environment, select relevant tables, and transform the data for reporting.

The analysis was built using selected tables related to customers, orders, invoices, suppliers, products, warehouse operations, and financial transactions.
